# ✅ To-Do List Application

A simple and responsive To-Do List web application built using HTML, CSS, and JavaScript.  
This application helps users manage daily tasks with add, complete, delete, and local storage functionality.


## 🚀 Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Data saved using Local Storage
- Press Enter to add task
- Responsive design for mobile, tablet, and desktop
- Clean and simple user interface


## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons

## ⚙️ Functionality

### Add Task
Users can enter a task and add it to the task list.


### Complete Task
Click on the check button to mark a task as completed.


### Delete Task
Click on the delete button to remove a task.


### Local Storage
Tasks remain saved even after refreshing the page using browser Local Storage.
